1. Executive Summary

背景・問題
仕様書v3で「Shopify Admin Embedded App方式」を明確に指定していたにも関わらず、前回実装でHydrogenルート（admin.reviews.tsx等7ファイル）として誤って構築。CEOから「Hydrogenは使わない」明示指示があり、全面的な再構築が必要。
本プロジェクトの目的
口コミ機能を Shopify Admin Embedded App として完全に再構築し、Shopify管理画面サイドバーから「口コミ管理」項目で直接アクセス可能にする。お客様向け投稿フォームも Hydrogen から剥離し、Embedded App 配下の公開ルートで提供。
再利用可能な既存資産（廃棄しない）
① Shopify Metaobject 6 定義（DB スキーマ相当）
② Shopify Flow ワークフロー（注文発送→14日後トリガー、id 01KRJ1JF33PNW1ENMTQZ584SN5）
③ Astromeda Product Manager アプリ（client_id 0410f99d…）
④ SHOPIFY_APP_AUTOMATION_TOKEN / GitHub Actions CI/CD 配管
⑤ ドメイン知識（既往のロジック設計）
技術スタック（Shopify公式推奨）
Framework: Shopify Remix App Template（@shopify/shopify-app-remix）
UI: Shopify Polaris + App Bridge v4
Hosting: Fly.io（Shopify公式ドキュメント推奨）
Auth: Session Token (App Bridge) / OAuth2
Email: Shopify Flow + Shopify Email（10,000 通/月 まで無料・既存 Flow id 01KRJ1JF33PNW1ENMTQZ584SN5 を流用）
CI/CD: GitHub Actions（既存配管を流用）
